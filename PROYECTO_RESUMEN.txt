PROYECTO: CONVERTIDOR DE ARTÍCULOS A AUDIO MP3
===============================================

ESTADO: ✅ COMPLETADO Y PROBADO EXITOSAMENTE

Fecha de Finalización: 15 de noviembre de 2025


RESUMEN
-------

Se ha desarrollado un programa completo que convierte artículos web en archivos
de audio MP3. El sistema extrae contenido de URLs, procesa el texto y lo convierte
a voz usando tecnología de Google Text-to-Speech.


ARCHIVOS PRINCIPALES
--------------------

1. texto_a_voz.py (~250 líneas)
   - Módulo principal con todas las funciones
   - Interfaz interactiva
   - Manejo robusto de errores
   - Documentación completa con docstrings

2. README_texto_a_voz.md
   - Guía de instalación y uso
   - Ejemplos de código
   - Solución de problemas
   - Referencia completa de funciones

3. ejemplos_uso.py
   - 6 ejemplos prácticos diferentes
   - Menú interactivo para seleccionar ejemplos
   - Casos de uso variados

4. audios/ (carpeta)
   - Almacena los archivos MP3 generados
   - Contiene: prueba_python.mp3 (archivo de prueba exitoso)

FUNCIONES PRINCIPALES
---------------------

extraer_articulo(url)
  └─ Extrae contenido de un artículo desde URL

procesar_texto(texto)
  └─ Procesa y limpia el texto en párrafos

convertir_texto_a_voz(texto, nombre_archivo, idioma, velocidad_lenta)
  └─ Convierte texto a voz y guarda como MP3

convertir_articulo_a_audio(url, nombre_salida, idioma, velocidad_lenta)
  └─ Flujo completo: extrae, procesa y convierte

interfaz_usuario()
  └─ Interfaz interactiva para uso directo


CÓMO USAR
---------

1. Ejecución interactiva:
   python texto_a_voz.py

2. Uso en código:
   from texto_a_voz import convertir_articulo_a_audio
   ruta = convertir_articulo_a_audio("https://...", "nombre", "es")

3. Ver ejemplos:
   python ejemplos_uso.py

LIMITACIONES Y NOTAS
--------------------

- Requiere conexión a Internet
- gTTS tiene límites de velocidad
- Artículos muy grandes pueden tomar más tiempo
- Algunos sitios pueden bloquear web scraping
- La calidad de voz depende de los servidores de Google
